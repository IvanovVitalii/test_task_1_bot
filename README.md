# Тестовое задание:
Написать простого бота для мессенджера Телеграмм.
На вход бота подается имя домена. Например, «Yandex.ru».  
В ответ бот присылает сообщение, в котором содержится:
- [X] дата регистрации данного домена,
- [X] кол-во дней до продления,
- [X] наличие SSL сертификата: да/нет,
- [X] если SSL сертификат найдет, то сколько дней осталось до его продления.

Мне не важно на каком языке вы будете писать бота.
Ссылку на бота, инструкцию по использованию и исходники присылаем на почту ________, тема письма 
«Телеграм-бот».
